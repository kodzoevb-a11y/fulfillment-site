# Сайт фулфилмента полного цикла

Статический one-page сайт на русском языке для склада полного цикла под Wildberries, Ozon и Яндекс.Маркет. Проект можно разместить на обычном хостинге без сборки.

## Структура

- `index.html` — разметка лендинга, SEO-мета, форма, модальное окно политики.
- `assets/css/styles.css` — адаптивный UI, сетки, анимации, мобильная липкая кнопка.
- `assets/js/main.js` — меню, FAQ, модальное окно, валидация формы, уведомления.
- `assets/img/favicon.svg` — favicon.
- `assets/img/warehouse-work.jpg` и `assets/img/warehouse-shelves.jpg` — локальные изображения склада.

## Запуск

Можно открыть `index.html` напрямую в браузере.

Для проверки через локальный сервер:

```bash
python3 -m http.server 8123
```

После запуска сайт будет доступен по адресу `http://localhost:8123`.

## Что нужно заполнить вручную

В открытом поиске не удалось надежно подтвердить телефон, адрес склада и реквизиты, совпадающие между источниками. Поэтому в `index.html` оставлены TODO-комментарии и видимые плейсхолдеры:

- подтвержденный телефон;
- ссылка WhatsApp только после подтверждения номера;
- адрес склада;
- юридическое наименование, ИНН, ОГРН/ОГРНИП;
- оператор персональных данных в политике конфиденциальности;
- реальный обработчик формы заявки.
